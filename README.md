## Sobre a Asynchronous Work

 - Acontece no momento.
 - Não precisa persistir após as reinicializações do  app ou do dispositivo.
 - Ocorre fora da linha de execução principal ou bloqueia a linha de execução principal.

 - Kotlin: Corrotinas
